## 🚀 How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/Side_projects.git
   cd Side_projects
   ```

2. Navigate to any project:
   ```bash
   cd contact_book
   python contact_book.py
   ```

3. Each project includes:
   - Python code
   - A local `README.md` with instructions
   - Optional: dependencies or data files
